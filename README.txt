O projeto prático da disciplina Fundamentos de Sistemas Operacionais consiste em implementar um "pseudo-sistema operacional", 
isto é, um programa que realiza uma abstração das tarefas de um SO convencional - o gerenciamento de processos, de memória, 
de arquivos e de entrada/saída. O programa foi desenvolvido na linguagem Python versão 3.9.x e os arquivos de texto para 
input (files.txt e processes.txt) encontram-se no subdiretório test.

Para a execução do programa, no diretório raiz do trabalho deve ser inserido o seguinte comando em seu terminal:
$ python3 main.py
